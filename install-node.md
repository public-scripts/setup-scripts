## Usage

### Install latest LTS Node.js

```bash
chmod +x install-node.sh
./install-node.sh
```

---

### Install specific version

#### Example: Node.js 22

```bash
./install-node.sh 22
```

#### Example: Node.js 20

```bash
./install-node.sh 20
```

---

### Verify Installation

```bash
node -v
npm -v
```
